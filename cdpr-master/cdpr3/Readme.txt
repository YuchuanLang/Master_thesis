This project uses ROS Jade and Gazebo 5 so it is advised to use this on the ubuntu platform

You can start up the robot using $ roslaunch cdpr3 cube.launch

The robot has a model plugin to demonstrate its task space dynamic control possibility.

You can start the normal controller by using $ roslaunch cdpr3 controlnormal.launch

You can start the scurve controller by using $ roslaunch cdpr3 controlnormal.launch

To change the robot, we can launch $rosrun rqt_gui rqt_gui

You can also change the project and improve it as you want
